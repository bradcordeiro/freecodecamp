URL Shortener Microservice

Example #1:

HTTP GET:

https://treasure-fibre.glitch.me/new/https://www.freecodecamp.com/challenges/url-shortener-microservice

Response:

{
    "original_url":"https://www.freecodecamp.com/challenges/url-shortener-microservice",
    "short_url":"https://treasure-fibre.glitch.me/0"
}

Visiting https://treasure-fibre.glitch.me/0 will redirect to:

https://www.freecodecamp.com/challenges/url-shortener-microservice

Example #2:

HTTP GET:

https://treasure-fibre.glitch.me/new/https://expressjs.com/en/api.html

Response:

{
    "original_url":"https://expressjs.com/en/api.html",
    "short_url":"https://treasure-fibre.glitch.me/2"
}

Visiting https://treasure-fibre.glitch.me/2 will redirect to:

https://expressjs.com/en/api.html
